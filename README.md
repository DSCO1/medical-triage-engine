# 🩺 NabzAI – AI-Powered Clinical Triage & Intelligent Care Recommendation System

<div align="center">

### Intelligent Symptom Analysis • Specialist Recommendation • Follow-up Planning • Appointment Scheduling

An AI-inspired Clinical Decision Support System that analyzes patient symptoms using a hybrid rule-based reasoning engine and recommends the appropriate specialist, urgency level, follow-up plan, and appointment details through an interactive dashboard.

---

## 📌 Overview

NabzAI is an intelligent clinical triage engine designed to simulate the initial patient assessment process.

The system accepts patient symptoms, severity, and duration as input and performs:

- Symptom normalization
- Clinical rule evaluation
- Risk assessment
- Specialist recommendation
- Confidence scoring
- Follow-up care planning
- Dummy provider matching
- Appointment scheduling

The project demonstrates how AI-inspired decision systems can assist healthcare professionals by reducing triage time and improving routing efficiency.
